[2번째 과제 - 1월 5일부터 - 진행 이전]

Dockerfile과 docker-compose.yml은 미리 작성되어 있으므로 도커 설치 후 명령어 입력만으로 바로 빌드와 컨테이너 생성이 가능할 것입니다.
 - 각자의 이름 (영어)로 path명을 지정하여, 아래 2개의 라우터를 만들어 보세요.
  - /(이름) : json 응답으로 {"message" : "(이름)"}
  - /(이름)/html : template 폴더 내의 example.html을 참고하여, 간단하게 자신을 소개하는 페이지를 만들어 보세요. (jinja2 템플릿을 사용할 겁니다.)
